# cubic-bezier-in-js
An Implementation of css ```cubic-bezier``` function in js
## Definition
Cubic Bezier is a function that takes two Points:
  <br>
    ```P0 = (x0, y0)```
    <br>
    ```P1 = (x1, y1)```
    <br>
    returns a function that goes like
    ```(t: [0 - 1]) => [0 - 1]```
    <br>
    <br>
    ```t = 0``` is the start of animation
    ```t = 1``` is the end of animation
    <br>
